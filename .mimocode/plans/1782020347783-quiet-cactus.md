## File Changes

### 1. `app/Http/Controllers/DashboardController.php` (BARU)

```php
public function index()
{
    return Inertia::render('Dashboard', [
        'stats' => [
            'total_olts' => Olt::count(),
            'total_onus' => Onu::count(),
            'active_onus' => Onu::where('status', 'active')->count(),
            'active_sessions' => AuditSession::where('status', 'active')->count(),
            'scans_today' => OltHistory::where('action', 'Scan')
                ->whereDate('created_at', today())->count(),
            'total_actions' => OltHistory::count(),
        ],
        'recent_activity' => OltHistory::with(['user', 'olt'])
            ->latest()->take(5)->get()
            ->map(fn($h) => [
                'id' => $h->id,
                'created_at' => $h->created_at->format('d M Y H:i'),
                'user' => $h->user?->name ?? 'N/A',
                'olt' => $h->olt?->name ?? 'N/A',
                'action' => $h->action,
                'status' => $h->status,
            ]),
        'onu_breakdown' => Onu::selectRaw('status, count(*) as total')
            ->groupBy('status')->get(),
    ]);
}
```

### 3. `resources/js/pages/Dashboard.vue` (REWRITE TOTAL)

Struktur:
- `defineProps` menerima `stats`, `recent_activity`, `onu_breakdown`
- **Row 1**: 4x Card stat (grid 4 kolom) - masing-masing pakai icon Lucide + angka + label
- **Row 2**: 2x Card stat (scan hari ini, total aksi)
- **Row 3**: 2 kolom grid:
  - Kiri: Card "Aktivitas Terakhir" - `<table>` 5 baris
  - Kanan: Card "Status ONU" - progress bar CSS per status + jumlah
- **Row 4**: Quick Actions - 4 tombol Button link ke halaman utama

Komponen shadcn-vue yang dipakai:
- `Card`, `CardHeader`, `CardContent`, `CardTitle`, `CardDescription`
- `Badge` (untuk status action: success/failed)
- `Button` (quick actions)

Icon Lucide yang dipakai:
- `Server`, `Cpu`, `CircleCheck`, `ClipboardCheck` (stat cards)
- `Scan`, `History` (stat cards row 2)
- `ArrowRight` (quick action buttons)
- `Network` atau `MonitorDot` (ONU breakdown)

### 4. Tambah route controller di `app/Providers/RouteServiceProvider.php` atau `bootstrap/app.php`?

Tidak perlu - controller di-register otomatis di `routes/web.php`.

## Color Scheme untuk Status ONU
- `active` вҶ’ hijau (`text-emerald-600`, bar bg emerald)
- `registered` вҶ’ biru (`text-blue-600`, bar bg blue)
- `unconfigured` вҶ’ amber (`text-amber-600`, bar bg amber)
- `inactive` вҶ’ abu (`text-muted-foreground`, bar bg gray)
